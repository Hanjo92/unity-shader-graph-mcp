# 1.0.0 Final Checklist

## Goal

Promote the current `1.0.0-rc.1` package-backed engine to final `1.0.0` without reopening broad engine scope.

## Definition Of Done

- The current package-backed editing engine remains stable on Unity 2022.3 with Shader Graph 17.3.0.
- Live MCP transport binding is finalized enough for real external tool calls.
- Release packaging and installation flow are documented from a fresh checkout.
- The verified runtime boundary is clearly documented so users understand what is and is not guaranteed.

## Must Finish Before 1.0.0

- Finalize live MCP transport binding for the Python server.
- Validate end-to-end external tool invocation against a real MCP client.
- Run a fresh-project install smoke test from the tagged release.
- Confirm the current verified node and connection matrix still passes in Unity EditMode tests.
- Confirm release docs and install docs reflect the actual shipped product shape.

## Should Finish Before 1.0.0

- Clean up top-level setup instructions for Unity package import and server startup.
- Tighten error messages around unsupported node and port combinations.
- Freeze the supported matrix language in docs so the 1.0 boundary is explicit.
- Capture one recommended happy-path demo flow:
  - `create_graph`
  - `add_property`
  - `add_node`
  - `connect_ports`
  - `save_graph`

## Explicitly Not Required For 1.0.0

- Universal all-node support.
- Universal all-port connection support.
- Template-backed graph creation beyond the current blank-graph path.
- Broad expansion into every texture, UV, and custom-function workflow.

## Recommended Release Shape

- `1.0.0`
  - stable package-backed Shader Graph engine
  - live MCP transport binding
  - documented installation and verified runtime boundary
- `1.1.x`
  - broader node and connection matrix
  - texture and UV workflows
  - deeper authoring ergonomics

## Final Pre-Tag Pass

- Run Unity `EditMode` tests to full pass.
- Run the highest-value debug menus on a fresh blank graph.
- Verify package version and Python version metadata.
- Confirm [CHANGELOG.md](/Users/song/Projects/unity-shader-graph-mcp/CHANGELOG.md) includes all release-facing changes.
- Draft GitHub release notes before tagging.
